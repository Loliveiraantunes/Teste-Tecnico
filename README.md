# SumOne-Gestao-Biblioteca
Este é o Teste Técnico para vaga de Programador Full-Stack.

# Arquitetura
  O Projeto foi desenvolvindo seguindo a arquitetura em MVC(Model, View e Controller), juntamente com o básico de Orientado a Objeto. 

# Requisitos

    Linguagem: 
      Java EE 7 WEB
    
    Servidor:
      Apache Tomcat ou TomEE ou GlassFish

# Bibliotecas
    JSF 2.2
    JSTL 1.2.1
    JDK 1.8

# Instruções
  Após terminar de configurar o ambiente do Projeto, basta executar o projeto ou o arquivo "index.jsp" com o Servidor Tomcat ou Glassfish.
  O Programa irá inicializar a tela mostrando os acervos Exibidos no casos de usos e os Inputs com os campos de filtragem, baste     selecionar os itens que queira listar e clicar em procurar. 

OBS: Não sei como aplicar Injeção de Dependencia de forma isolada para as bibliotecas. Sorry! ;-;
